--------------------Build Instructions--------------------
1) Download and install Maven 3

2) Make sure the Maven executables are on the PATH system variable

3) Go to the save-the-world directory

4) Run this command:	
	mvn clean package

5) "target" subdirectory gets created. It contains the built application.
	
--------------------How to Run the Software--------------------
-> Build the software.

-> Go to the bin directory.

-> To start the server, run 
	server.bat	... on Windows
	server.sh	... on Unix, Linux
If you wish to specify program parameters, open the script in a text editor
and change the relevant values. 

-> To start the client in terminal, run
	console_client.bat	... on Windows
	console_client.sh	... on Unix, Linux
If you wish to specify program parameters, open the script in a text editor
and change the relevant values.

-> To start the client in graphical mode, run
	gui_client.vbs	... on Windows
	gui_client.sh	... on Unix, Linux
